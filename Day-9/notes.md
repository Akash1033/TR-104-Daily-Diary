# Day 9 – Conditional Rendering

Today I learned how to display UI conditionally in React.

Sometimes we want to show different UI depending on conditions like login status or data availability.

React allows conditional rendering using JavaScript logic.

Common approaches include:

if statements  
ternary operators  
logical AND operator

For example, we can show a welcome message if the user is logged in and a login button if they are not.

This concept is very useful for building dynamic applications.
