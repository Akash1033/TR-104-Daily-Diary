# Day 6 – React Project Structure

Today I spent some time understanding how a React project is organized.

When we create a React project using Vite, several folders and files are generated automatically. At first it looked confusing, but after exploring each file, it started making sense.

Important files I noticed:

index.html  
This is the main HTML file where the React app is injected.

main.jsx  
This is the entry point of the application. It connects the React app to the HTML file.

App.jsx  
This is the main component where most of the UI logic starts.

The src folder contains most of the application code such as components, styles, and logic.

Understanding the folder structure helped me see how different parts of a React application are connected.
