# Day 2 – Components and JSX

Today I learned about JSX and components.

JSX stands for JavaScript XML. It allows us to write HTML-like syntax inside JavaScript.

Example:

const element = <h1>Hello World</h1>;

React components are reusable pieces of UI.

Example of a simple component:

function Welcome() {
  return <h2>Welcome to React</h2>;
}

Components help keep the code organized and reusable.

Today I practiced creating multiple components and rendering them inside App.jsx.
