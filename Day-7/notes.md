# Day 7 – Creating Multiple Components

Today I practiced creating multiple components and organizing them properly.

Instead of writing everything inside App.jsx, I created separate component files like:

Navbar.jsx  
Footer.jsx  
Card.jsx

Then I imported them into App.jsx.

This makes the project easier to manage because each component handles a small part of the UI.

For example, the Navbar component only focuses on navigation while the Footer handles the bottom section of the page.

Breaking the UI into components is one of the most powerful ideas in React.
