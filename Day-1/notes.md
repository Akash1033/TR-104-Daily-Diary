# Day 1 – Introduction to React

Today I started learning React.

React is a JavaScript library used to build user interfaces. It helps developers create fast and interactive web applications.

One important concept in React is **components**. Instead of writing the entire UI in one file, we break it into smaller reusable pieces called components.

Examples of components in a website:
- Navbar
- Sidebar
- Product Card
- Footer

React also uses something called **Virtual DOM**. Instead of updating the entire DOM, React compares changes and only updates the necessary parts. This improves performance.

Today I also created my first React project using Vite.

Commands used:

npm create vite@latest
npm install
npm run dev

I explored the project structure and understood the purpose of important files like `App.jsx` and `main.jsx`.
