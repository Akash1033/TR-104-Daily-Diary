# Day 8 – Props in Detail

Today I explored props in more depth.

Props allow a parent component to send data to a child component.

Instead of hardcoding values inside components, we can pass dynamic values using props.

For example, if we have a product card component, we can pass different product names and prices through props.

This makes components reusable because the same component can display different data.

I also practiced destructuring props, which makes the code cleaner and easier to read.
