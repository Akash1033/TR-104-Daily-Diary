# Day 10 – Lists and Keys

Today I learned how to render lists of data in React.

When we have multiple items such as products, users, or messages, we usually store them in arrays.

React allows us to render arrays using the map() function.

While rendering lists, React requires something called a key.

Keys help React identify which items changed, were added, or removed.

Using keys improves performance and helps React update the UI efficiently.
